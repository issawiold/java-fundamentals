## **the routes for the website**
- **`/signup` (GET)**: renders a page with forms for you to signup at .
- **`/signup` (POST)**: POST THE USERNAME AND PASSWORD in a postgres database .
- **`/login` (GET)**: renders a page with forms for you to log into your account.
- **`/` (GET)**: Renders a page with two buttons one leads to a login page while the other leads to a signup page.
- **`/mainpage` (GET)**: Renders a page with the username written in it .


Do note that all post requests of this application are handled via forms

Do not under any circumstances try to access the post requests via any other method
